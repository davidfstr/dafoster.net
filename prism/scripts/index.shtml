Printing Scripts

These printing scripts allow you to print to the GT Library, the GT Student Center, Central Printing, and the College of Computing (CoC).


Download GTPrint Printing Scripts v2.0.1
Does not work with the new printing system introduced Fall 2008.
Allows you to print to the GT Library, the GT Student Center, and Central Printing from your print dialog. Printing in both portrait and landscape orientations is supported.

Printing to the College of Computing

To setup printing to the CoC on Mac OS X 10.4 (Tiger):
  1. Open Printer Setup Utility.
  2. Click the Add button on the toolbar. A dialog will appear.
  3. Click the IP Printer button on the toolbar.
  4. From the Protocol menu, select HP Jet Direct - Socket.
  5. For the address/name/location/printer model, specify one of:
rosalyn wormwood
130.207.14.144 130.207.14.145
GT CoC - Rosalyn GT CoC - Wormwood
CoC 107, Thin Client Lab CoC 107, Thin Client Lab

LAWN Autologin Script

This script automatically logs you into LAWN, Georgia Tech's wireless network, whenever your computer connects to it.

Windows users should try the LawnLogin program written by Ethan Trewhitt.

Download Login to GTWireless v1.5 for Leopard (Mac OS X 10.5)
Automatically logs into LAWN when your computer connects to it.

Automatically logs into LAWN when your computer connects to it.
